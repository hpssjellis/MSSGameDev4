9 Programming Components

  1. Input/Output: .value / .innerHTML | draws in/puts out values

  2. Storage: variables x = 4 | stores/takes in values

  3. Decisions: if(x = 4){myTemp = 4} | if statements that activate code if defined conditions are met



  4. Loops: for(let i=0; i=x; i++){myTemp += 'Loop'} | for statements that loop code

  5. Functions: function myFun(x,y){return x*y} | named code

  6. Events: onClick="{alert('hi')}"> | activates code



  7. Arrays: myAr = new Array(4); myAr[2] = 8 | numbered variables

  8. Objects: x = {age:20, name:'fred'} | named variables

  9. Classes: methods-->functions, properties-->variables | can be a bundle of everything
